# Summitline Outfitters - Seasonal Apparel Return Policy

**Document ID:** kb-1
**Category:** Returns

Summitline Outfitters accepts returns on seasonal apparel within **60 days**
of purchase with the original receipt or order confirmation email. Items
must be unworn, unwashed, and have all original tags still attached.

## Exceptions

- Sale-final items marked "Final Sale" are not eligible for return.
- Custom embroidery and altered items are not eligible for return.
- Items damaged through misuse, normal wear, or improper laundering are not
  eligible for return but may qualify for the warranty program (see the
  lifetime warranty policy for eligible product lines).

## How to return

1. Initiate the return at `summitlineoutfitters.com/returns` with your order
   number.
2. Print the prepaid shipping label included with the return confirmation
   email.
3. Drop the sealed package at any authorized carrier location.

Refunds are issued to the original payment method within 5 business days
of receipt at our Boulder, CO fulfillment center.
