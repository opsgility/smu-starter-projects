# sample_data/

Drop two optional files here for the Exercise 2 and 3 smoke tests:

- `storefront.jpg` — any JPEG. Used by `/vision-ask` in Exercise 2 Step 6 and by `test_client.py`. A photo of any storefront is fine; the exercise is about the request shape, not the image content.
- `invoice.pdf` — any simple invoice PDF. Used by `/extract-doc` in Exercise 3 Step 5 and by `test_client.py`.

If either file is missing, `test_client.py` will skip that endpoint with a note instead of failing.

`question.wav` (for `/voice`) is generated by the snippet in Exercise 3 Step 4 and lands at the repo root, not here.
