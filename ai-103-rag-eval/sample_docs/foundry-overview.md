# Microsoft Foundry overview

Microsoft Foundry is the unified Azure AI platform for building, deploying, and governing
generative applications and AI agents. It combines a model catalog (LLMs, small language
models, multimodal, code), a project workspace, and tools for retrieval-augmented generation
(RAG), evaluation, and observability.

## Key concepts

- **Hub** — the parent resource that holds shared infrastructure and governance.
- **Project** — a workspace under a hub for a specific application or team.
- **Connections** — references to data sources (Azure AI Search, Storage), other Cognitive
  services, or third-party APIs.
- **Deployments** — model endpoints provisioned within a project.
- **Agents** — long-running, tool-augmented assistants built on the Agent Service.

## Regions

Foundry is generally available in the East US, East US 2, West Europe, North Europe,
Southeast Asia, and Australia East regions, with new regions added regularly.
