# AI-901 Lesson 6 — Single-Agent Client Starter

Scaffold for Lesson 6. In the lab you'll:

1. Create an agent in the Foundry portal (or via `create_agent.py`).
2. Save the resulting `AGENT_ID` to `.env`.
3. Run `agent_client.py` to chat with the agent via the Agents SDK.

## Files

- `create_agent.py` — optional: create the agent programmatically (instead of using the portal)
- `agent_client.py` — create a thread, send a message, wait for a run, print the reply
- `tools.py` — sample function-tool definitions you can attach to the agent
