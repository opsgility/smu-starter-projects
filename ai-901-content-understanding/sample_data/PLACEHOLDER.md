# Sample data

Content Understanding works best with real documents/images/audio. This folder
ships with small stand-ins so you can run the stubs end-to-end; swap in
realistic assets when you do the exercise.

Suggested files to drop here during the exercise:
- `invoice.pdf` — any short (1–2 page) invoice PDF.
- `receipt.jpg` — a phone photo of a store receipt.
- `voicenote.wav` — 10–20 s of spoken text (use `ai-901-speech` to record one).

The default `invoice.md` is a tiny markdown stand-in for the very first run.
