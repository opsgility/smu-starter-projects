# AI-901 Lesson 1 — Responsible AI Starter

Scaffold for the Lesson 1 lab exercises in the **AI-901: Microsoft Azure AI Fundamentals** course. You'll implement the TODOs in `main.py` to score text against the six Responsible AI principles using Azure AI Content Safety.

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env   # fill in your Content Safety endpoint + key
```

## Run

```bash
python main.py
```

## Files

- `main.py` — Content Safety client scaffold with TODO markers
- `sample_data/texts.json` — sample texts representing each RAI principle
- `rubric.md` — the RAI evaluation rubric used by the exercise
