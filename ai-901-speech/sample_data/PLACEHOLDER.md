# Sample audio

Drop a small `.wav` file named `hello.wav` in this folder before running
`speech_to_text.py --file sample_data/hello.wav`.

The exercise walks you through two easy ways to get one:

1. Run `python src/text_to_speech.py` first to create `output.wav`, then copy it here as `hello.wav`.
2. Use the Windows Sound Recorder (or `ffmpeg -f dshow -i audio=...`) to record a short phrase.

Keep the file under 100 KB / a few seconds — this is a unit-scale sample, not production data.
