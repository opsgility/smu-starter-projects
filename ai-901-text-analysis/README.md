# AI-901 Lesson 7 — Text Analysis Starter

Scaffold for Lesson 7. You'll build a small text-processing tool that combines:

1. **Azure AI Language** for key-phrase extraction, entity recognition, and sentiment analysis.
2. **Foundry-deployed chat model** for summarization (generative text analysis).

## Files

- `analyzer.py` — entry point
- `language_calls.py` — Azure AI Language SDK wrappers
- `summarize.py` — Foundry chat model wrapper
- `sample_data/reviews.txt` — product reviews to analyze
