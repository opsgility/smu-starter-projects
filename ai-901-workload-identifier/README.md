# AI-901 Lesson 3 — Workload Identifier Starter

Scaffold for the Lesson 3 lab exercises. You'll build a small CLI that takes a business scenario and maps it to the correct Azure AI workload + service.

## Workloads covered

- Generative AI / Agentic AI
- Text analysis
- Speech
- Computer vision
- Information extraction (documents, images, audio, video)

## Run

```bash
pip install -r requirements.txt
python identifier.py "Our support team wants to auto-summarize inbound tickets."
```
