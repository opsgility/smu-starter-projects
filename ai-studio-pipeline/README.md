# AI Studio Pipeline — Starter Project

This starter project supports the **GCP: AI Studio to VS Code** exercises.

## Structure

```
lesson02/
  exercise01.py   # Completed: freeform prompt via SDK
  exercise02.py   # Completed: structured few-shot classifier
  exercise03.py   # Completed: chat with system instructions
  exercise04.py   # Completed: export and run AI Studio code
lesson04/
  exercise01.py   # TODO: system instructions runner
  exercise02.py   # TODO: model comparison benchmark
  exercise03.py   # TODO: PromptPipeline class
  exercise04.py   # TODO: pipeline with error handling and retry
```

## Setup

```bash
pip install google-generativeai
```

Your `GOOGLE_API_KEY` is already set in the environment.
