# RetireScope — Lesson 1 Starter

Empty Next.js 16 scaffold. You'll add the design system and sidebar nav in Lesson 2.

## Getting Started

```bash
npm install
npm run dev
```

In another terminal:

```bash
web-tunnel
```

Open the public URL the tunnel prints. You should see the default Next.js page.

## What's Included

- Next.js 16 with App Router and Turbopack
- React 19 + TypeScript 6
- Tailwind CSS 4 (no design tokens defined yet — you'll add them)
- shadcn baseline utilities (`clsx`, `tailwind-merge`, `lucide-react`)

## What You'll Build

Lesson 1 teaches the architecture. Lesson 2's exercises walk you through:

1. Defining design tokens in `app/globals.css` via Tailwind 4's `@theme` block
2. Forcing dark mode and wiring Geist + JetBrains Mono fonts
3. Building the sidebar navigation
4. Stubbing the five core routes (Dashboard, Accounts, Projection, Tax Planner, Assistant)

When you finish Lesson 2, your code will look like the `claude-finance_02` snapshot.
