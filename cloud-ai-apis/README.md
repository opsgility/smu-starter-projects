# Cloud AI APIs — Starter Project

This starter project supports the **GCP: Building a Multi-API Content Analyzer** exercises.

## Structure

```
lesson02/
  exercise01.py   # Completed: Vision API label + OCR detection
  exercise02.py   # Completed: Natural Language sentiment + entity analysis
  exercise03.py   # Completed: Translation API detect + translate
  exercise04.py   # Completed: combined image content analyzer
lesson04/
  exercise01.py   # TODO: Vision API wrapper class
  exercise02.py   # TODO: Natural Language API wrapper class
  exercise03.py   # TODO: Speech-to-Text transcription
  exercise04.py   # TODO: ContentAnalyzer pipeline combining all APIs
```

## Setup

```bash
pip install requests
```

Your `GOOGLE_API_KEY` is already set in the environment.
