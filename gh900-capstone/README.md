# Capstone: Open-Source Project

Build a complete open-source project on GitHub demonstrating mastery of all GitHub Foundations concepts.

## Requirements

### Repository Setup
- Public GitHub repository with descriptive name, topics, and description
- Proper README.md, LICENSE, .gitignore
- SECURITY.md, CODE_OF_CONDUCT.md, CONTRIBUTING.md

### Git Workflow
- Feature branches following GitHub Flow
- Meaningful commit messages with conventional format
- At least 3 feature branches merged via pull requests

### Collaboration
- At least 5 GitHub Issues with labels, milestones, and assignees
- Issue templates and PR templates configured
- CODEOWNERS file for review assignment

### CI/CD with Actions
- At least one GitHub Actions workflow (build/test)
- Status badge displayed in README
- Workflow triggers on push and pull_request

### Project Management
- GitHub Project board with custom fields
- Issues organized across board columns
- Use of Table or Roadmap view

### Security
- Branch protection on main (require PR, require reviews)
- Dependabot enabled
- No secrets committed to repository

### Community
- Release with proper semantic versioning and release notes
- Social preview image
- Repository configured for community contributions

## Evaluation Rubric

| Dimension | 1 (Beginning) | 3 (Proficient) | 5 (Exemplary) |
|-----------|---------------|-----------------|----------------|
| Repo Setup | Missing key files | README + LICENSE + .gitignore | Full community health files |
| Git Workflow | Single branch only | Feature branches used | Clean history, conventional commits |
| Collaboration | No issues | Issues with labels | Templates, milestones, CODEOWNERS |
| Actions CI | No workflows | Basic build workflow | Build + test + badge |
| Project Mgmt | No project board | Board with columns | Custom fields, automation, views |
| Security | No protections | Branch protection | Full security config |
| Community | No releases | Tagged release | Semver + notes + social preview |

**Total: 35 points (7 dimensions x 5 points each)**
- 30-35: Exemplary
- 22-29: Proficient
- 15-21: Developing
- Below 15: Needs additional practice
