# GitHub Community & Open Source

Practice contributing to and managing open-source projects on GitHub.

## Tasks

1. Fork an existing repository
2. Create community health files:
   - CODE_OF_CONDUCT.md
   - CONTRIBUTING.md
   - Issue templates and PR templates
3. Make a contribution via fork and pull request
4. Add topics, description, and a social preview image
5. Create and manage releases with release notes
6. Explore GitHub Sponsors and community profiles
