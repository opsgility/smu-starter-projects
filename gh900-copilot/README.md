# GitHub Copilot & AI-Powered Development

Explore GitHub Copilot features and AI-assisted development.

## Exploration Scenarios

### Scenario 1: Code Completion
- Open a new file and write function signatures
- Observe how Copilot suggests implementations
- Accept, modify, or reject suggestions

### Scenario 2: Comment-Driven Development
- Write descriptive comments and let Copilot generate code
- Compare: "// sort array ascending" vs "// sort array of objects by date field descending"

### Scenario 3: Test Generation
- Write a function and ask Copilot to generate unit tests
- Evaluate the test coverage and edge cases

### Scenario 4: Code Explanation
- Use Copilot Chat to explain unfamiliar code
- Ask Copilot to suggest improvements or find bugs

### Scenario 5: Documentation
- Use Copilot to generate docstrings and README content
- Compare AI-generated docs with manually written ones
