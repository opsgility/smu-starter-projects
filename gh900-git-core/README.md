# Task Manager тАФ Git Core Starter

A simple JavaScript task manager app for practicing Git branching and merging.

## Files

- `app.js` тАФ Task manager with `addTask` and `listTasks` functions

## What You Will Do

In the exercises you will:
- Create feature branches with `git branch` and `git switch`
- Add a `completeTask` function on a `feature/complete-task` branch
- Add a `deleteTask` function on a `feature/delete-task` branch
- Perform fast-forward merges and three-way merges
- Resolve merge conflicts between branches
- Visualize branch history with `git log --oneline --graph --all`

The starter app has `addTask` and `listTasks` тАФ you will add more functions on branches and merge them back into main.
