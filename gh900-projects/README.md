# GitHub Projects & Project Management

Create and manage a GitHub Project board from the TODO items in this codebase.

## Tasks

1. Create GitHub Issues from the TODOs in Program.cs
2. Create a new GitHub Project (Board view)
3. Add issues to the project and organize by status
4. Create custom fields (Priority, Sprint, Estimate)
5. Set up automation rules for issue transitions
6. Use Table and Roadmap views
