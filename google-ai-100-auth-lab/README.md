# google-ai-100-auth-lab

Starter for **Google AI 100 — Lesson 2: Service Accounts & ADC in Practice**.

You will practice three auth patterns:

1. Service account key file (`GOOGLE_APPLICATION_CREDENTIALS`)
2. `gcloud auth application-default login` (user ADC)
3. Observing the error signatures when permissions or API enablement are wrong

## Files

- `verify.py` — runs a minimal Gemini call; prints success or the exact failure
- `requirements.txt`
