# google-ai-100-copy-drafter

Starter for **Google AI 100 — Lesson 5: Policy-Aware Copy Drafter**.

You will use system instructions, safety settings, structured output, and
thinking mode to produce on-brand catalog copy for Nimbus Outfitters.

## Files

- `draft.py` — CLI skeleton
- `tone_of_voice.md` — brand voice rules
- `content_policy.md` — content rules
- `requirements.txt`
- `THINKING.md` — blank file for your observations in the thinking-mode exercise
