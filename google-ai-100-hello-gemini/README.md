# google-ai-100-hello-gemini

Starter for **Google AI 100 — Lesson 1: Your First Gemini API Call**.

You will complete `main.py` to call Gemini on the Gemini Enterprise Agent Platform using the unified `google-genai` SDK.

## What's in the box

- `main.py` — skeleton with `TODO` markers
- `requirements.txt` — pins `google-genai`
- `.env.example` — template for environment variables (copy to `.env`)

## Quick check

Your workspace should already have a Python venv activated with `google-genai` installed. Run:

```bash
python -c "from google import genai; print(genai.__version__)"
```

If that prints a version number, you are good to go.
